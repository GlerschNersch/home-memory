About

Home Memory — a narrative intelligence layer for Home Assistant.
Connects to your HA instance via WebSocket with live event streaming. Uses a local Ollama model to narrate, explain, and coach your smart home.
Ollama CORS setup: Start Ollama with
OLLAMA_ORIGINS=* ollama serve
or set your exact HA origin. Required for browser fetch to work.
